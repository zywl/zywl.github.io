正式安装HEXO

Node和Git都安装好后，可执行如下命令安装hexo：

npm install -g hexo

初始化

然后，执行init命令初始化hexo到你指定的目录，我是直接cd到目标目录执行hexo init的。命令：

hexo init

它会提示你执行： npm install

这一步一定要执行不然后面会出错！

当然也可以随意一个盘符里的文件夹执行这个命令

好啦，至此，全部安装工作已经完成！
